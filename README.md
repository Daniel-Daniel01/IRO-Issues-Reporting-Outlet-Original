# IRO — Issues Reporting Outlet

A simple front-end demo of a community issue reporting system, built with plain HTML, CSS, and JavaScript (no frameworks, no backend).

## Pages
- `index.html` — Home page
- `login.html` — Login (UI only)
- `register.html` — Register (UI only)
- `dashboard.html` — User dashboard with stats and quick actions
- `report.html` — Submit an issue (with validation)
- `reports.html` — My Reports table (filter + search)
- `admin.html` — Admin dashboard demo

## How to view
Just open `index.html` in any web browser. No installation or server needed.

## Notes
- All data (users, reports) is dummy/demo data — nothing is actually saved.
- Forms use simple JavaScript validation for required fields.
- Built to be responsive across desktop, tablet, and mobile.
